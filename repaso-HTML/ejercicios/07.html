Ejercicio 07: listas ordenadas y no ordenadas

Esta será una página para crear una lista ordenada(ol), es decir, con un número a la izquierda de cada elemento(li).
El título de la página será Listas Ordenadas.
Como primera linea aparecerá un titular de nivel 1(h1) con el texto Orden de salida.
En el cuerpo de la página tendremos la lista con el orden de llegada de una carrera(ol): un número y los nombres que se indican abajo:

  1. María, dorsal 102
  2. Luisa, dorsal 104
  3. Ana, dorsal 169
  4. Carmen, dorsal 308

Orden de salida:

  1. María, dorsal 102
  2. Luisa, dorsal 104
  3. Ana, dorsal 169
  4. Carmen, dorsal 308

Ejemplo de html:

            <h1>Orden de salida:</h1>
            <ol>
              <li>María, dorsal 102</li>
              <li>Luisa, dorsal 104</li>
              <li>Ana, dorsal 169</li>
              <li>Carmen, dorsal 308</li>
            </ol>
