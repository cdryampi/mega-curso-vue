Ejercicio 04: enlaces o hipervinculos

En la primera línea le pondremos un titular de nivel 1 (h1) con el texto Enlaces (a) a secciones (section).

El cuerpo de la página contendrá tres párrafos(p), cada uno contendrá un enlace a la url(a) que se indica a continuación con el texto que figura a su lado

url: /tutorhtml5/07b-etiquetas-formato.htm, Etiquetas de formato

url: /tutorhtml5/04-head.htm, Sección head

url: /tutorhtml5/05-body-html5.htm, Sección Body

Es decir se verá una lista de tres enlaces. Comprueba que funcionan.

Etiquetas de formato / Ejercicio 1

Sección head / Ejercicio 3

Etiquetas de formato / Ejercicio anterior
